# laflamme-zachary-a3-survivalproject
This is my 2D game assignment where you dodge the balls!

INSTRUCTIONS
WASD / Arrow keys to maneuver
SHIFT to go slower
SHIFT + movement keys to juggle your hitbox!
SPACE to jump. You can also hop on the walls!

Don't touch the red, or it's game over.

SIDENOTE: There is no collision. Not yet!
